# Roadmap phases (condensed)

Full: blueprint **Phần 08**. Gates: [release-gates.md](../shared/release-gates.md).

| Phase | Goal | Exit (short) |
|---|---|---|
| **0** (2–4w) | Prep: market/broker, ToS, schemas, ADR min set | ADRs signed; risk register; schema v1 subjects |
| **1** (4–6w) | MVP: Core+Data+Backtest+rule strategy+Dashboard+KS+obs; paper | ≥30d paper; Phase 1 checklist PASS |
| **2** (6–8w) | AI + small live (≤5% NAV) | ≥4w live in budget; 0 system KS; L3 game-day; restore drill |
| **3** (6–8w) | Auto-retrain, MRM, shadow/canary, Approvals, GitOps | ≥3 canary promotes; OTel ≥95%; no deploy-fault rollbacks |
| **4** (ongoing) | Multi-strategy / builder / scale; multi-user only after legal | Legal sign-off before 3rd user |

Principle: paper until backtest + shadow stable; live capital starts small.
