# Feature Spec: PAPER-FE-POLISH

## Overview

Owner activated `PAPER-FE-POLISH`: strategy list UX + prefill Account ID. No Deferred, no live.

## Requirements

- FR-001: Persist last Account ID in sessionStorage after Accounts create / Strategies use.
- FR-002: Prefill Strategies create + list; auto-load list when prefilled.
- FR-003: Prefill Dashboard + Alerts Account ID fields from the same store.
- FR-004: Clearer strategy list status presentation (active / paused / stopped).
